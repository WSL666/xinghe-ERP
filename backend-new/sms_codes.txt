# 用法:
#   每行写一个验证码, 格式: 任意非空字符串(建议 6 位数字, 如 123456)
#   用户在前端输入该验证码注册, 用过后整行自动删除
#
# 维护方式:
#   1. 你随时编辑本文件, 追加一行验证码即可
#   2. 把这一行验证码发给客户
#   3. 客户用过后, 后端会自动从文件里删掉这一行(原子写入)
#
# 说明:
#   - 以 # 开头的行为注释, 不会被当作验证码, 也不会被删除
#   - 空行会被忽略
#   - 验证码区分大小写, 首尾空格会被自动去掉
